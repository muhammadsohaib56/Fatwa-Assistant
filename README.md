# Fatwa Assistant

A web-based chatbot that provides Islamic Q&A (Fatwa) answers based on the Quran, Hadith, and user-selected Fiqh schools.

## Prerequisites

- Python 3.8+
- A Gemini API key (or similar AI API key)
